# Sun Clock – Legacy release

**Legacy ☀️ APK distribution**

This repository provides the **`a5`** build line of _Gordon's Sun Clock_ for devices that cannot use the main release line (**Android 5–6** or **armeabi-v7a**). Keeping legacy and main releases in separate repositories ensures clean update notifications in tools such as Obtainium (available on [GitHub](https://github.com/ImranR98/Obtainium) and F-Droid) and avoids unnecessary update prompts for users who stay on the legacy line.

For more information, see the main repository: https://github.com/gaxmann/gordonssunclock
